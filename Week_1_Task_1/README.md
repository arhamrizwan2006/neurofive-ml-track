# Neurofive ML Track — Week 1

Exploratory Data Analysis on the Titanic dataset, done as part of the Neurofive Solutions Machine Learning Fundamentals internship track.

## Files
- `week_1_setup.ipynb` — notebook with the EDA
- `train.csv` — raw dataset from Kaggle

## What I did
- Loaded the dataset and inspected it using `.head()`, `.tail()`, `.shape()`, `.info()`, `.describe()`
- Checked for missing values and duplicates
- Identified which columns are numerical and which are categorical
- Wrote a short data story summarizing the findings

## Dataset
Titanic - Machine Learning from Disaster from Kaggle.

## Tools used
Python, pandas, NumPy, Jupyter Notebook
